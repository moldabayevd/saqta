# Исследование: Qwen3.5-Omni-Plus как замена Whisper

**Дата:** 22-23 апреля 2026  
**Цель:** проверить, даёт ли Qwen3.5-Omni-Plus (DashScope API) заметный выигрыш в качестве транскрипции по сравнению с локальным `Whisper-large-v3-russian` (antony66 fine-tune) на типичной часовой встрече Казахтелекома (русская речь, термины связи, имена коллег с казахскими корнями).

## Сетап

- **Эталон:** часовая запись инновационного комитета (61 мин, mp4, 11 ГБ)
- **Whisper:** `ggml-large-v3-russian.bin` + Silero VAD + initial prompt со словарём имён/терминов
- **Qwen3.5-Omni-Plus:** DashScope API, MP3 64kbps, чанки по 10 минут (лимит data-uri 10 МБ)
- **Sequence-similarity:** `difflib.SequenceMatcher.quick_ratio` на нормализованном тексте (без пунктуации, без таймкодов, lowercase)

## Результаты

| Метрика | Whisper-russian (local) | Qwen3.5-Omni-Plus (API) |
|---|---|---|
| Слов чистого текста | 6 874 | 7 368 |
| Чистых символов | 41 КБ | 43 КБ |
| Покрытие речи | базовое | **+7%** (поймал больше) |
| Время на час встречи | **5 мин** | 7 мин (+ кост резки) |
| Стоимость | бесплатно | **~$0.30** |
| Греет ноут | да (Metal на полную) | нет (всё в облаке) |

**Сходство текстов: 96.7%**  
**Различия: 3.3%**

### Где Qwen3.5-Omni выигрывает

- **Сложные термины:** «Smallworld» одним словом, «машинночитаемом» правильно, «ЦППСР» вместо размытого «ППСР»
- **Микро-диалоги:** ловит короткие реплики на стыках («Простите, можно секунду? — Да, я вас слушаю») — Whisper их размазывает или сливает
- **Пунктуация и регистр:** естественнее, без обрывов фраз
- **Нет повторов на тишине:** даже без VAD не зацикливается

### Где Whisper не уступает

- Имена в этой записи — оба распознали одинаково (Сергей, Марат, Канат, Татьяна Викторовна)
- Цифры (47 млн тенге, 16%, приказ № 236) — оба точно
- Покрытие основного смысла встречи — идентичное (~97% совпадение)

## Вывод

Для **рутинных рабочих встреч** Whisper-russian + словарь покрывает ~97% задачи. Дополнительные 3% от Qwen3.5-Omni-Plus не оправдывают $0.30/встреча × N встреч/месяц, если транскрипт затем всё равно скармливается LLM для саммари (LLM нивелирует мелкие огрехи Whisper).

Смысл переключаться на Qwen3.5-Omni возникает в трёх случаях:
1. Запись с шумом или плохим микрофоном (Plus устойчивее к деградации)
2. Юридический/медицинский транскрипт, где важно каждое слово дословно
3. Корпоративный сценарий с собственной GPU-инфрой (в Казахтелекоме — 16× H200), куда можно поднять открытые веса Qwen3.5-Omni-Light/Flash через vLLM и получить тот же результат бесплатно для всей команды

## Локальные альтернативы, которые проверялись

- **Qwen3-ASR-1.7B локально (через `qwen-asr` пакет, MPS):** работает, но на M4 Pro в 8-10 раз медленнее `whisper.cpp` (час аудио ≈ 50 минут vs 5 минут). Качество — между Whisper и Qwen3.5-Omni. **Не оправдывает скорости.**
- **NVIDIA Parakeet TDT v3:** не тестировался, по бенчмаркам слабее Qwen3-ASR при аналогичной скорости.
- **Whisper v4:** не существует, OpenAI прекратили open-релизы Whisper (последний — large-v3 ноябрь 2023).

## Решение

- **Дефолт:** `Whisper-large-v3-russian` локально (текущее).
- **Pipeline:** запись → Whisper → ручной/Claude-саммари. Локальные LLM (qwen3:14b) для саммари выкинуты — слишком греют ноут, качество ниже Claude API на ~15-20%.
- **Будущее:** когда выйдут GGUF-кванты Qwen3.5-Omni Light для llama.cpp / MLX (через 1-2 недели после релиза 22.04.2026) — переоценить как локальный «всё-в-одном» (audio → текст → саммари в одном проходе).
