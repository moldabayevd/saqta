# 🎉 KT Recorder v0.1.0 — Initial Release

**Дата релиза:** 2026-04-23
**Тег:** `v0.1.0`
**Размер:** ~50 KB кода (только bash + Python обёртки)

---

## TL;DR

Приватный локальный рекордер встреч для macOS с поддержкой казахского языка
и kk/ru code-switching. Whisper под капотом, Markdown на выходе, Obsidian-
совместимо, всё локально. Ноль подписок, ноль облака.

## ✨ Что внутри

### Запись и транскрипция
- 🎙️ Запись встреч через QuickRecorder (хоткей `⌘⇧R`) с разделёнными
  аудио-дорожками (микрофон + системный звук отдельно)
- 🤖 Транскрипция через [whisper.cpp](https://github.com/ggml-org/whisper.cpp)
  с Metal-ускорением на Apple Silicon
- 🇷🇺 Whisper-large-v3-russian fine-tune (antony66) для русского — заметно
  точнее vanilla large-v3 на наших именах и терминах
- 🇰🇿 **Казахский язык + kk/ru code-switching** через Qwen3-ASR-1.7B
  (опциональный бэкенд) — основная фишка под рынок Казахстана
- 🎚️ Silero VAD для борьбы с галлюцинациями на тишине
- 📚 Initial prompt словарь имён и терминов — whisper не коверкает
  «Smallworld», «ДСА-11», корпоративные названия

### Автороутер языков
- 🌍 `scripts/transcribe-auto.sh` определяет язык по первым 30 секундам
  и автоматически выбирает бэкенд:
  - `ru` / `en` → whisper-large-v3 (без изменений вашего пайплайна)
  - `kk` / `mix` → Qwen3-ASR (специально под code-switching)

### Интерфейс
- 🎨 Красивая TUI-менюшка `scripts/kt` на базе [gum](https://github.com/charmbracelet/gum)
  со спиннерами, превью файлов и адаптивным меню
- 📋 Список записей со статусами: 🔴 raw / 🟡 transcribed / 🟢 summarized
- 🖱️ Двойной клик ярлычок на рабочем столе через
  `launchagents/create-desktop-shortcut.sh` — никакого терминала для
  обычной работы

### On-demand vs Auto
- 🔋 По умолчанию **on-demand режим** — ничего не крутится в фоне, мак не
  греется. Запустил меню → выбрал запись → пошла обработка
- 🔄 Опционально: LaunchAgent + fswatch для автообработки новых записей

### Экспорт
- 📝 Markdown с YAML frontmatter (Obsidian-совместимо)
- 🎬 VTT-субтитры с таймкодами (открывается в IINA/VLC)
- 🎭 Транскрипт с разделением по ролям через две дорожки:
  `**Я:** ... **Собеседники:** ...`

## 📊 Бенчмарки

На реальной часовой встрече Казахтелекома (русская речь, корпоративные
термины, имена с казахскими корнями):

| Метрика | Whisper-russian (наш дефолт) | Qwen3.5-Omni-Plus (API, $0.30) |
|---|---|---|
| Сходство текстов | базовый | **96.7%** |
| Время на час | 5 мин | 7 мин |
| Стоимость | бесплатно | ~$0.30 |
| Греет ноут | да | нет (облако) |

**Вывод:** разница 3.3% не оправдывает $$$, локальный пайплайн закрывает
~97% рутинных задач.

## 🛠 Технические детали

- **Язык:** Bash (основа), Python (только Qwen3-ASR обёртка)
- **Зависимости:** whisper-cpp, ffmpeg, fswatch, terminal-notifier,
  quickrecorder, gum, ollama (опц.), jq
- **Платформа:** macOS 13+ (Apple Silicon рекомендуется)
- **CI:** GitHub Actions с shellcheck для всех bash-скриптов
- **Лицензия:** MIT
